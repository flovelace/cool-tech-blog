# Cool Tech Blog
## Purpose
A tech blog that allows users to sign up, login, and logout. Users can write posts about their tech thoughts, and comment on the tech thoughts of other users.

## Built with
* HTML
* CSS
* JavaScript
* Express.js
* Node.js
* Bcrypt
* Bootstrap
* Handlebars
* mysql2
* Sequelize
